# CP

A personal repository for Competitive Programming practice, focused primarily on **Codeforces**, with **NeetCode / LeetCode** used to strengthen Data Structures & Algorithms fundamentals.

This repository serves as a long-term archive of progress, learning, and consistent problem solving.

---

## Primary Focus

- **Codeforces** for contest practice, speed, and problem-solving under pressure  
- **NeetCode / LeetCode** for learning core DSA patterns and implementation clarity

---

## Objectives

- Improve algorithmic thinking and contest performance  
- Build strong fundamentals in Data Structures & Algorithms  
- Develop speed, accuracy, and debugging ability  
- Maintain consistency through regular practice  
- Track long-term growth through commits and solved problems

---

## Contents

- Codeforces practice and contest solutions  
- NeetCode / LeetCode topic-wise solutions  
- Standard algorithm templates  
- Notes, observations, and learnings  
- Reference resources
